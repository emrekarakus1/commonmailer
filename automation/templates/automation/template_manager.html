JSON Format Guide

Use this JSON structure to create templates. You can use placeholders like {name}, {company} etc.

Example JSON Structure:

{
  "Welcome": {
    "subject": "Welcome {name}!",
    "body": "<p>Hello {name},<br><br>Welcome to our service!</p>"
  },
  "Invoice": {
    "subject": "Invoice for {company}",
    "body": "<p>Dear {name},<br><br>Please find your invoice attached.</p>"
  }
}
